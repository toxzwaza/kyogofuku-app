### Phase 2：DBバックアップ（thumbnail カラム追加のため必須）

```bash
# .env から DB情報取得
DB_NAME=$(docker compose exec -T app grep '^DB_DATABASE=' .env | cut -d= -f2 | tr -d '\r')
DB_USER=$(docker compose exec -T app grep '^DB_USERNAME=' .env | cut -d= -f2 | tr -d '\r')
DB_PW=$(docker compose exec -T app grep '^DB_PASSWORD=' .env | cut -d= -f2 | tr -d '\r')

# shared_db（MariaDB 10.11）から dump
cd /srv/laravel-apps/shared
docker compose exec -T db mariadb-dump -u $DB_USER -p"$DB_PW" $DB_NAME \
  > /tmp/event_reserve_backup_$(date +%Y%m%d_%H%M%S).sql

ls -lh /tmp/event_reserve_backup_*.sql | tail -3
```
