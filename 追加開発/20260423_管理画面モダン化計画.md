# 管理画面モダン化計画（2026-04-23〜）

## 1. 概要

| 項目 | 内容 |
|------|------|
| 立案日 | 2026-04-23 |
| 指示元 | 村上 |
| 対象アプリ | kyogofuku-app（京呉服平田 イベント予約システム） |
| 対象範囲 | 管理画面（`/admin/*`）＋共通基盤（Layout・Components・Tailwind） |
| 対象外 | 公開ページ（`resources/js/Pages/Event/*`、LP、予約フォーム、LIFF画面） |
| 開発優先度 | 高（デザイン刷新＋業務効率化機能を同時に実現） |

### 目的

機能仕様は維持したまま、以下を達成する。

1. **デザインシステム化** - Tailwind直書きから脱却し、トークン化されたモダンUIへ
2. **情報設計の強化** - ダッシュボード、タイムライン、KPI可視化で意思決定を高速化
3. **業務効率化** - 一括操作、エクスポート、リマインダー自動化で日常業務を短縮

### 技術スタック（現状）

- Laravel 9.19 / Inertia.js 1.0 / Vue 3.2.41 / Tailwind 3.2.1 / Vite 4.0
- 共通コンポーネント 約35個（`resources/js/Components/`）
- FullCalendar 6 / Chart.js 4 / Swiper 12 / Fabric.js 6 導入済み
- アイコンライブラリ未導入／ダークモード未対応／カラートークン未整備

### 進捗記号

`[ ]` 未着手　／　`[~]` 着手中　／　`[x]` 完了　／　`[-]` スキップ・保留
