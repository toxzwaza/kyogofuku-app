### 修正1：予約フォームに個人情報取扱同意チェックボックスを追加
フォームタイプ（振袖／袴）に関わらず、予約フォームの末尾（送信ボタン直上）に「個人情報の取り扱いに同意する」チェックボックスを固定配置。デフォルトはチェック済み。チェックを外すと送信ボタンが非活性化される。

### 修正2：開催会場ブロックの開催日を当日以降のみ表示
イベント詳細ページ（`Event/Show.vue`）の開催会場カード内「開催日」表示を、当日より前の日程は除外。残日程が無い会場では「開催日」ブロック自体を非表示化。

### 修正2：開催日フィルタ

```javascript
function formatVenueDates(dateStrings) {
  if (!dateStrings || dateStrings.length === 0) return [];
  const now = new Date();
  const todayStr = `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}-${String(now.getDate()).padStart(2, '0')}`;
  const sorted = [...dateStrings].sort().filter((d) => d >= todayStr);
  if (sorted.length === 0) return [];
  // ...既存の月またぎ・連続日フォーマット処理...
}
```

- 当日（ローカル時刻の YYYY-MM-DD）は表示に含める（以降）
- 全て過去日の会場は「開催日」ブロックそのものを非表示
- 既存の連続日（`〇, 〇` / `〇 ~ 〇`）・月またぎ表記ロジックは維持
